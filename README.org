* Personal Emacs Configuration
** Installation
- [[https://github.com/d12frosted/homebrew-emacs-plus][Emacs Plus]]
  #+begin_src sh
  brew tap d12frosted/emacs-plus
  brew install --cask emacs-plus-app@master
  #+end_src
** Prerequisites
- ripgrep
  #+begin_src sh
  brew install ripgrep
  #+end_src
- Zed Mono
  #+begin_src sh
  brew install --cask font-zed-mono-nerd-font
  #+end_src
- .authinfo
  #+begin_src sh
  machine api.openai.com login apikey password <password>
  machine api.x.ai login apikey password <password>
  machine generativelanguage.googleapis.com login apikey password <password>
  #+end_src
** Errors
*** ~linker library load error~
#+begin_src sh
brew reinstall gcc libgccjit
#+end_src

#+begin_src sh
emacs -Q & \
find-file init.el & \
